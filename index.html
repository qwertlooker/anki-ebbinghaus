设置

AI 大模型设置（优先使用）

配置 AI API 后，自动制卡将优先使用 AI 生成高质量单词卡片。支持 OpenAI 兼容接口。

⚠️ 注意：DeepSeek API 支持浏览器直接调用；OpenAI、NVIDIA、通义千问等 API 因 CORS 限制，需要通过代理服务器调用。若 AI 调用失败会自动降级到词典 API。

Redis 设置

在网页内填写 Redis REST URL 与 Token。推荐使用 Upstash Redis REST。页面打开和操作前后都会优先执行同步。

当前方案把 Token 存在浏览器本地，适合个人使用或小范围私有部署。若要面向多人公开访问，建议改成 serverless 代理层后再暴露网页。
